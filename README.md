# Steering-to-Safety-Inference-Alignment-of-Transformers-using-Probes-and-GSAEs
Inference-time alignment of LLMs using Linear Probes and Gated Sparse Autoencoders (GSAEs) on a frozen RoBERTa backbone. No retraining required. Discovers interpretable safety-relevant latent atoms, trains steering vectors, and evaluates synergistic probe+SAE interventions on toxicity &amp; jailbreak benchmarks.
